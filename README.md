# MarketingHubb

The best digital marketing agency in Varanasi. This is a Next.js website showcasing our services, courses, and contact information.

## Features

- 4 pages: Home, About, Services, Contact
- Responsive design with Tailwind CSS
- Smooth animations with Framer Motion
- Contact form
- Floating WhatsApp and call icons
- Professional color scheme

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To build the project for production:

```bash
npm run build
```

## Technologies Used

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons

## Project Structure

- `app/` - Next.js app directory
  - `components/` - Reusable components
  - `page.tsx` - Home page
  - `about/page.tsx` - About page
  - `service/page.tsx` - Services page
  - `contact/page.tsx` - Contact page
